Práctica 2.2: Uso de selectores

En esta página hay ejemplos pensados para comprobar reglas CSS con distintos selectores.

Bloque A: selectores básicos

Subtítulo h3 (parece un título como h2, pero no lo es)

A1. Esto tiene clase destacado (fondo rosa claro).

A2. Esto tiene clase destacado-importante (NO es destacado).

A3. id principal-texto (NO es principal).

Bloque B: selectores combinados

B1. Soy un p con id importante (tamaño 200%).

B2. Soy un div con id importante-box (parece relacionado, pero NO cumple).

Bloque C: selectores descendientes

C1. Nota dentro de div (cursiva).

C2. Nota fuera de div (NO cursiva).

C3. Nota dentro de zona-es (roja y cursiva).

C4. Nota dentro de zona-especial (cursiva, pero NO roja: la clase no coincide).

Bloque D: agrupación y combinación de clases

D1. Clase aviso (subrayado).

D2. Clase avisos (NO es aviso).

D3. Tiene urjc y reglamento (AND: aplica).

D4. Solo urjc (NO cumple también reglamento).
